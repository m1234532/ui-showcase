# 🎨 UI 风格展示库

探索前沿设计趋势，激发创意灵感。

## 展示的风格

### 🫗 Glass Morphism（玻璃质感）
- 半透明背景配合 backdrop-filter 模糊
- 细边框营造层次感
- 适合现代、清新的界面

### 🧈 Neumorphism（新拟态）
- 柔和的内外阴影创造凹凸效果
- 极简配色，依靠光影塑造形态
- 适合工具类、设置界面

### 🌌 Aurora Background（极光渐变）
- 流动的渐变网格动画
- 多色径向渐变叠加
- 适合背景、氛围营造

### 🍱 Bento Grid（便当盒布局）
- 模块化卡片系统
- 响应式自适应
- 适合仪表板、功能展示

### ✨ Micro-interactions（微交互）
- 细腻的悬停、点击反馈
- 缩放、涟漪、位移动画
- 提升用户体验
