# 🎨 UI 风格展示库

探索前沿设计趋势，沉浸式体验每一种风格。

## 页面结构

- **首页** - 白色主题导航，快速进入各个风格页面
- **Glass Morphism** - 玻璃质感（按钮、卡片、表单）
- **Neumorphism** - 新拟态（按钮状态、开关、输入框）
- **Aurora Background** - 极光渐变（多种配色方案）
- **Bento Grid** - 便当盒布局（仪表板、功能展示、瀑布流）
- **Micro-interactions** - 微交互（悬停、点击、加载动画）
