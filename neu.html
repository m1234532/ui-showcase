按钮状态

凸起、按下、凹陷等多种按钮状态

/* 凸起效果 */
.neu-raised {
  box-shadow: 8px 8px 16px #a3b1c6, -8px -8px 16px #ffffff;
}

/* 按下效果 */
.neu-pressed {
  box-shadow: inset 6px 6px 12px #a3b1c6, inset -6px -6px 12px #ffffff;
}

/* 悬停时切换状态 */
.neu-btn:active {
  box-shadow: inset 4px 4px 8px #a3b1c6, inset -4px -4px 8px #ffffff;
}
